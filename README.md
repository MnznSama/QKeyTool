# QKeyTool

一个通过模拟QQ快速登录协议来获取相关凭据的Python工具。

- 自动获取QQ登录所需的token和cookies
- 支持多账号选择
- 自动处理登录跳转流程
- 计算并获取必要的认证参数

## 注意事项

- 仅支持已登录的QQ客户端环境
- 需要本地QQ客户端配合使用
- 请确保网络连接正常
